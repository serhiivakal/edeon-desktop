# Bioisostere Library Methodology

The Edeon Bioisostere Engine leverages three complementary sources for generating bioisosteric suggestions:
1. **SwissBioisostere**: Derived rules from the SwissBioisostere database.
2. **mmpdb**: Matched Molecular Pairs derived from public ChEMBL approved drugs.
3. **Manual Curation**: High-confidence classical bioisosteres from medicinal chemistry textbooks.

## Citations
- SwissBioisostere: Wirth et al. (2013) Nucleic Acids Res.
- mmpdb: O'Boyle et al. (2016) J. Chem. Inf. Model.
